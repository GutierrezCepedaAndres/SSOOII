PARA LA EJECUCION Y COMPILACION DEL CODIGO SE REALIZARAN LAS SIGUINTES INTRUCCIONES

1- Se realizara un make all para la creacion y compilacion de los archivos necesarios.

2- Se realizara make test1 para la ejecucion de un ejemplo de prueba.

3- Se ejecutara make clean para la eliminacion de archivos y directorios de trabajo

4- Para la ejecucion del codigo con otros archivos se ejecutara la siguiente linea de comando

    g++ ssooiigle.cpp -o ssooiigle -pthread -std=c++11

    esto compilara el programa

    ./ssooiigle <nombre_fichero> <palabra> <numero_hilos>

    para la ejecucion del programa.
